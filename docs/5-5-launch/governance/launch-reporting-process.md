# Launch Readiness Reporting Process (PM-10)

## Reporting Artifacts

1. Daily scorecard summary.
2. Weekly launch-readiness brief.
3. Gate status report before any candidate release.

## Daily Summary Template

1. Benchmark score delta.
2. Reliability delta (crash-free sessions, recovery success).
3. Security findings state.
4. P0/P1 defect count.
5. Top 5 blockers and owners.

## Weekly Brief Template

1. Dimension-by-dimension threshold status.
2. Must-have feature completion status.
3. Risk register delta.
4. Release gate projection (`on track` / `at risk` / `blocked`).
