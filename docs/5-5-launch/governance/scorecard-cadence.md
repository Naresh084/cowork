# Weekly Scorecard Cadence (PM-05)

## Cadence

1. Every Monday 10:00 local: publish weekly scorecard.
2. Every Wednesday 16:00 local: mid-week regression checkpoint.
3. Every Friday 17:00 local: release-gate readiness review.

1. Updated benchmark scorecard.
2. Trend deltas week-over-week.
3. Regression list with owners and due dates.
4. Release-gate pass/fail summary.
